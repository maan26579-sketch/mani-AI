# Ukraine–Russia Conflict — Slide Deck (Google Slides–ready)

This folder contains a Google Slides–ready slide source for a 10-slide class presentation (Title + 8 presenter slides + Conclusion/Sources). The content is formatted for easy copy/paste into Google Slides. Images are referenced by URL (Wikimedia Commons). A small CSV file with chart data is included.

Files in this folder:
- slide_deck.md  — Full slide content (titles, bullets) and speaker notes
- chart_data.csv — Simple chart data for Slide 6 (Refugees vs IDPs)
- images_links.txt — Direct image URLs and attribution (download if you want local images)
- README.md — Instructions: how to import into Google Slides and produce a .pptx

---

## Notes
- The slides include speaker notes timed for ~60–90 seconds per presenter.
- Image URLs are provided in `images_links.txt`. You can insert images by URL in Google Slides or download and upload them.
- If you want, I can later generate a .pptx file and push it to this repo — tell me and I will create the binary and provide it in a way I can commit.
